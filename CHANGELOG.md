# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [Unreleased]

### Added
- Initial release of iPhoto Viewer
- Modern Liquid Glass UI design with dark/light themes
- iPhone connectivity via USB and WiFi
- Automatic device discovery using Bonjour/mDNS
- Virtualized photo grid for high performance
- Drag-and-drop photo transfer functionality
- Custom frameless window with integrated controls
- Theme toggle button in title bar
- Multi-select photo support
- Batch photo operations
- Real-time connection status
- Responsive layout for all screen sizes

### Technical Features
- Built with Electron + React + TypeScript
- Framer Motion for smooth animations
- React Window for virtualized lists
- Context-based state management
- Optimized rendering and caching
- Memory-efficient photo loading

## [1.0.0] - 2025-12-25

### Initial Release
- First public release of iPhoto Viewer
- Core functionality for iPhone photo management
- Cross-platform support (Windows focus)
